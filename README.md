SAP BTP Document AI Extractor

This project is an end-to-end AI-powered document processing application built on SAP Business Technology Platform (BTP).

It allows users to upload PDF documents (invoices, receipts, purchase orders) and automatically extracts structured data such as:

Document number
Date
Vendor details
Total amount
Line items
🏗️ Architecture
SAPUI5 (Frontend)
Node.js CAP (Backend)
SAP Document Information Extraction (AI Service)
OAuth-based secure integration
⚙️ Features
PDF upload with validation
AI-based data extraction
Real-time status tracking using polling
Structured output (UI + JSON)
🚀 Use Case

Automates manual document processing and converts unstructured PDFs into structured, usable business data.
